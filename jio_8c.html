Read all lines from a file into an array of null-terminal strings.

The read lines include newline character(s), and are null terminated.

Parameters
filepathName of the file to open.
linesPointer to an array to store read lines in.
lengthsPointer to an array to store the length of read lines in.
nThe length of the lines and lengths arrays, which must be equal. If n is 0, lines and lengths will be allocated memory to begin with.
Returns
On success, returns the number of lines read (and hence the length of lines and lengths). On failure, returns -1 and errno is set to indicate the error.
Remarks
Lines and lengths should either be large enough to fit the entire file or able to be dynamically reallocated.
This function allocates a char* array for each line in the file. The programmer must free each entry in the lines array, then lines and lengths, when freeing allocated memory.
All allocations made are shrunk to the minimum size after reading in the file.

Errors

errno Description
EINVAL lines or lengths is a null pointer.
ENOMEM There was insufficient memory to expand either lines or lengths.
