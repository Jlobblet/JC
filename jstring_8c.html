iptr advance_line ( const char *  start,
const char **  end 
)

Read chars from start until reaching a newline, \0, or EOF.

Parameters
startPointer to the beginning of the string or line to read from.
endPointer to where to write the ending pointer of the string.
Returns
On success, return how many characters are between the start and end of the line. If no characters are read because start begins with \0 or EOF, return -1.
Remarks
The line is not guaranteed to be null-terminated. The end of the line is bounded by the pointer written to end.
