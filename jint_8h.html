◆ strtoi()

int strtoi ( const char *  nptr,
char **  endptr,
int  base 
)

Parses the C-style string nptr, interpreting its content as an integer of the specified base, which is returned as a value of type int. This function operates as a wrapper around strtol, discarding results outside the range of int.

Parameters
nptrC-style string containing the representation of an integer.
endptrPointer to a value to write the next character in nptr after the integer. If endptr is a null pointer it is not used.
baseNumerical base (aka radix) that determines valid characters and their interpretation. See strtol for details.
Returns
On success, the converted integer as an int value. If no valid conversion could be performed, 0 is returned. If the value read is out of range of an int, the function returns INT_MAX or INT_MIN, and errno is set to ERANGE.
